# 📊 DataInsight v2.0

**Transforme qualquer dataset em insights automáticos em minutos!**

Uma plataforma inteligente que processa QUALQUER tipo de dado (CSV, Excel) e gera:
- ✅ Análises automáticas
- ✅ Gráficos relevantes dinamicamente
- ✅ Relatórios profissionais em Excel
- ✅ Detecção de padrões e anomalias

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app_v2.py
```

Acesse: http://localhost:8501
